  Перед тем как распаковать этот архив с апдейтом в каталог,
где установлена старая версия V_KLay, удалите из него
папку "data" со всем ее содержимым. Иначе возникнет путаница с
лоадерами: появится несколько телефонов с одинаковыми именами.


  Before unpacking this archive with update to folder, were
old version of V_KLay exist, delete in it folder "data" and
all its subfolders.
